【原创软件】CQAPK工具箱一键安装冰箱黑域

2018年11月02日

# 简介

无论是冰箱还是黑域，其安装方法都过于麻烦，因此初琴制作了 CQAPK 工具箱，不仅可以一键安装冰箱和黑域，还可以不用手机助手安装 APK 文件到手机。

# 功能介绍

1. 支持右键安装 APK 到手机。
2. 支持一键安装冰箱和黑域
3. 支持一键查询手机 CPU 型号。

# 注意事项

0. 此程序未加密，请自行防止各种 BAT 病毒的侵蚀
1. 安装后不要修改或删除 D 盘下 ADB 文件夹
2. 卸载请使用安装包内选项 3 卸载

# 已知问题

不能安装目录或者文件名中有中文的 APK。
Windows10 下不能直接建立系统关联，需要右键 apk 文件选择默认打开方式。

由于初琴使用的打包工具的 BUG，不能在卸载时自动清理注册表，所以推荐卸载后使用 Wise Care 365 清理注册表。
