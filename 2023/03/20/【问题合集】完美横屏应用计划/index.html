【问题合集】完美横屏应用计划

2023年03月20日

# Q0: 模块怎么安装？

A0：本模块需要一定的动手能力，所以推荐先学会正常的 Magisk 模块安装方法后，再来尝试本模块。

# 关于 KernelSU：

下载地址中有一个文件夹是 KernelSU 的替换文件，替换到模块里就行了

# Q1: 我应该安装哪个版本？

A1：一共分为两个分支，每个分支有 5 个版本：

[正式分支]：合并了经过验证后确实可用的临时版本代码的分支，更稳定，适合大部分人使用。

[临时版本]：仅供反馈群内人员测试，更新频率非常高，说不准哪天更，但是 BUG 也非常多。

每个版本下的 5 个版本分别为：

1. 稳定 + 红米版 (推荐)：去除所有引起不稳定因素的适配。

安卓 11 下所有应用支持左右大小调节。

BUG 反馈请到这个群里反馈：205924955。目前已知问题如下：

# 关于酷安：

仅支持高于 13.0.0 版本的 APP，且不支持左右大小调节

# 关于微博 / 微博移动版：

仅支持高于 12.12.0 版本的 APP

# 关于百度网盘：

如果界面缺少按钮，则右上角高级设置里把默认进入全屏播放打开。感谢 @mikasaaaan 提供方案

# 关于知乎：

和知了模块有界面冲突，无解

# 关于 QQ 浏览器：

QQ 浏览器任务栏重叠是它自己的问题，没有解决方法

# 关于墨墨背单词：

最高支持 5.0.20

# 关于淘宝：

如果不能触发平行视界，需要清除应用数据然后重启应用，登录后再重启应用就好

部分应用有首次安装后闪退的问题，建议先关闭平行视界，登录，然后再开启平行视界，即可正常。如果不正常，则可以临时删除该应用的配置，参考 Q7。

X. 自用版：为作者自己用的版本，不保证在其他设备上的有效性。

​ 自用版包含了 6Max 移植包特有属性，如果需要在非 6Max 移植包的机器上刷，请移除模块中的 system 文件夹下的所有文件。

​ 包含所有对应用已经做了的适配，原则上 **不接受 BUG 反馈**。具体原因不想说了。目前额外调整如下：

# 关于酷安：

自用版最高支持酷安 12.0.5。只有自用版配合旧版酷安修复了竖屏分屏的问题

# 关于 TIM：

自用版在一定程度上仿照 QQ 的样式支持了平行视界，但是效果不好

# 关于网易云音乐：

自用版仅支持最新版的网易云音乐，是平行视界样式

# 关于微博：

自用版最高支持 12.12.3，再往上就是微博自己适配的 12L 接口了

# 关于微博移动版：

自用版最高支持 12.8.0，再往上就是微博自己适配的 12L 接口了

# 关于闲鱼：

闲鱼的逻辑不合理，这个我清楚，但是没办法。因为如果写成和淘宝一样的逻辑，就会导致左半屏幕卡死。所以我想出了用收藏页左右横跳实现刷新的办法，不合理，但是至少不会卡死

# 关于 CCTalk：

CCtalk 更新后，模块失效，要使模块生效，需删除 apk 中 AndroidManifest.xml 里的 <meta-data android:name=“embedded”; android:value=“true”/> 参数后，重新打包 apk 安装，请不要自动签名

# 关于喜马拉雅 FM：

官方适配太烂了，自用版里我移除了

# 其他额外的适配：

自用版额外适配了百词斩、白描、番茄免费小说、皮皮虾、小红书

安卓 11 下所有应用不支持左右大小调节，非常整洁（强迫症）。

2. 折叠屏版

折叠屏支持安卓 11 - 安卓 13 的所有版本的 MIUI For Fold

3.6Max 移植包专用版

6Max 移植包专用版用于支持 6Max 在各种机型上的移植包开启平行视界左右可调特性

分为 5 系版和 6 系版，仅限 6Max 移植包可用，其他版本都不可用！刷了会卡米！

4. 原生版：去除所有 MIUI 自己适配的代码，仅保留原生安卓 13 的适配。用于某些对系统有洁癖的玩家使用。

5. 卸载模块：用于卸载该模块。使用方法见 Q6。

版本之间可直接覆盖安装，无需先卸载再安装。

# Q2: 哪些应用支持强制平行视界？

A2：可以查看更新日志

# Q3: 哪些应用不支持强制平行视界？

A3：可以查看更新日志，其中标记为 [移除] 的，为官方已经适配或完全无法适配的应用。不支持强制平行视界的应用，以上应用可以通过 Q5 方法强制横屏。

# Q4: 哪些应用有更好的替代版本？

A4：大部分影音应用都有 TV 版，音箱版和车机版，请尽可能去下载这些版本，它们都完美适配了大屏设备。例如：

芒果 TV：推荐使用音箱版，下载链接

网易云音乐：如果害怕修改版有风险，建议使用车机版，下载链接

微博：推荐使用微博移动版，界面和功能都一样

# Q5: 本模块不支持的应用有没有办法强制横屏？

A5：有两个方法。

​ 方法 1：去酷安论坛搜索一个叫 [MaxMiPad] 的 LSPosed 模块，解锁竖屏限制，然后使用 [Rotation] 进行旋转。

​ 方法 2：使用名叫 [自由屏幕旋转] 的 LSPosed 模块，可以直接进行旋转。

​ 以上两个模块和本模块冲突，请勿同时对同一个应用使用。

# Q6: 如何卸载模块？

A6：求求了，卸载模块前，先在酷安 #完美横屏应用计划# 话题下留下卸载原因，要不我都不知道到底哪里做的不合各位心意了。

​ 本模块需要使用卸载模块卸载，卸载模块下载地址。

​ Step1：先刷入卸载模块，重启；

​ Step2：卸载卸载模块，重启；

​ Step3：一般到这里就完全恢复了，但是有少数用户并不会恢复。需要等待 MIUI 下发一次云控。通常需要 1 周左右的时间。

# Q7: 如何解决部分应用闪退 / 如何彻底关闭一个 APP 的适配？

A7：如果发现应用闪退，或者应用适配后效果还不如不适配，先在酷安 #完美横屏应用计划# 话题下发帖告诉我包名，我会在以后的版本中移除该应用。

​ 当应用闪退时，首先尝试关闭该应用的系统设置中的横屏模式开关，然后再打开登录。

​ 如果能正常登录，则可以尝试再开启横屏模式开关，一般就能正常使用了。如果开启横屏模式开关后依旧会闪退或工作不正常，则需要删除该应用在模块中的配置。

# 方法 1 (推荐，需要 PC)

​ 使用 PC 版模块修改器直接修改模块，移除对应应用的适配。

​ 模块修改器：下载地址

​ 模块修改器教程：在本博客下或者在我的哔哩哔哩空间内搜索教程视频。

# 方法 2 (手动删除，仅需要平板)

# 安卓 12 / 安卓 13

​ 使用 MT 管理器打开 /data/adb/modules/MIUI_MagicWindow+/common/product/etc/embedded_rules_list.xml。

​ 搜索要删除的应用的包名，定为到行，删除整行内容。保存文件，重启平板即可。

# 安卓 11

​ 使用 MT 管理器打开 /data/adb/modules/MIUI_MagicWindow+/common/system/users/0/magic_window_setting_config.xml。

​ 搜索要删除的应用的包名，定为到行，删除整行内容。保存文件，重启平板即可。

# Q8: 为什么重启后平行视界开关会自动打开？

A8：干掉云控的后遗症，解决方法参考 Q7。

# Q9: 为什么任务栏和小白条无法沉浸？

A9：官方不支持，我也没办法。

# Q10: 为什么从任务栏点进去的消息不能全屏？

A10：官方不支持，我也没办法 + 1。

# Q11: 我在哪里提交新的软件适配需求 / 提交我写的适配代码？

A11：在确定你的应用不在不可适配的范围内的情况下，可以在这个酷安的 #完美横屏应用计划# 话题下进行提交。或者加群 183114301 进行反馈。
