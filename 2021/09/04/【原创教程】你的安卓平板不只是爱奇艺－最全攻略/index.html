【原创教程】你的安卓平板不只是爱奇艺－最全攻略

2021年09月04日

前几天一个小伙伴发了一篇题目为《你的 iPad Air 不只是爱奇艺》的图文。感觉更多的情况是安卓平板被叫做 “爱奇艺” 平板。那么今天我就来分享一下，你的安卓平板除了 “爱奇艺” 还能干什么。

声明：我目前基本都是键鼠使用环境，没有使用手写笔的环境，所以手写笔和笔记软件相关内容请查看其他 UP 的推荐文。

文中有一些软件是和 iPad 共有的，iPad 用户也可以尝试搜索下载试用。

# 学习篇：

# 1.OfficeSuite

功能最多的 Office 全家桶，相比于微软官方的 Office，虽然兼容性略差，但是功能非常多。推荐在上面编辑自己私有的文档。如果是需要其他人打开的文档，不推荐用这个。

堪比电脑版 Office 的功能

# 2. 腾讯文档

支持多端 (PC，手机，平板) 多人同步编辑的云文档软件，真正的合作必备。以前和其他 UP 一起做视频文案时就用这个。现在主要用于不同平台手机、平板和电脑间同步文档。

支持简单的 Office 编辑

# 3. 思维导图 / Xmind

没什么介绍的，脑图画习惯了就是梳理思路的好东西。

脑图习惯了记录思路超舒服

# 4.iCircuit

电路模拟器，对于门电路学习，以及一些数字电路的模拟有奇效。

可以模拟一些简单电路和常见芯片

# 5.DroidEdit/C4Droid

大学 C 语言课必备的 C 语言编译器，可以在上面实现一些简单的代码的编写和运行。

支持语法高亮和编译

# 6.aid learning

安卓上的 LINUX 虚拟机，有了这个就可以在平板上搭建 Python 环境和 Java 环境，用于复杂的编程学习和脚本调试了。

完整的 LINUX

# 7.Fusion Apps

用于在手机上编写封装一些基于 H5 的小软件。图例是用该软件编写的小破站第三方客户端。

功能丰富

# 剪辑绘画篇：

# 1.Krita/PhotoPea

Krita 做为全量版的修图绘图软件，兼容支持 PSD 模式，是绘画，修图爱好者必备的 App。

Photopea 是 Photoshop 的网页版，也兼容支持 PSD，对于流量较多的用户可以使用这个。

Krita 功能堪比电脑版 Photoshop

Photopea

# 2.Adobe 全家桶

PhotoShop Express + LightRoom 都不用说了，大家都知道。

还有一款软件叫 PS Mobile，是一个支持多图层的 PhotoShop 官方版。遗憾的是只支持安卓 9 及以下。

支持多图层

# 3. 必剪 / 剪映

简单剪辑必备软件，都支持多视频轨道和多音轨。推荐使用鼠标和键盘！！

剪映没有任何会员机制，非常适合白嫖。而必剪由于绑定了 B 站激励，甚至不仅能白嫖还能赚钱。

必剪超好用，功能超级多

# 影音篇：

影音软件上，我相对更推荐 TV 版软件。做为非主要用于娱乐的平板，我对影音软件的要求是美观大于实用。

# 1.B 站 TV 版 / 腾讯视频 TV 版 / 优酷 TV 版

界面好看且适配才是王道，这界面 iPadOS 上的 HD 版本也没法比。

B 站 TV 版

# 2. 斗鱼 TV 版

比斗鱼 HD 更好看的布局，全面支持触屏，也支持弹幕。

上斗鱼 HD，下斗鱼 TV

# 3. 电视家

在平板上看电视直播其实也蛮棒的 。

流畅不卡

# 游戏篇：

游戏我大多还是在手机上打，所以只推荐一些我觉得在平板上有操作优势的游戏。

# 1. 红日

安卓上的红色警戒翻版，非常适合用触屏笔，一笔掌控千军万马。

操作手感挺好的 RTS 游戏

# 2. 炉石传说 / 命运之戒

炉石的平板版本和 PC 版的操作逻辑一致，更适合玩操作较多的卡组。记得在电脑模式下玩。

和电脑版一致的界面
其实类似于炉石操作的卡牌类游戏在平板上玩都很爽，比如命运之戒。

命运之戒

# 3. 少女前线 / 明日方舟

屏幕更大，更不容易点歪。

指挥官叛变了！

# 4.7723 游戏盒 / 葫芦侠三楼

下载游戏必备的游戏商店。在三楼可以下载很多 Steam 移植的大作以及其他主机平台和 PC 游戏。游戏玩家必备！

有很多 Steam 移植大作

# 系统加强篇：

主要用于补齐一些系统功能。

# 1. 简窗 / 米窗 / 悬浮通知

用于在不支持小窗模式的平板上开启任意 App 小窗。App 内有教程，按着操作就好了。

给某 App 开个小窗

# 2.Floating Apps/Mini Apps

用于在不支持上面三个软件的平板上，实现一些常用功能的小窗。

小窗模式的浏览器

# 3.ES 文件管理器

用于访问局域网内的 SMB/FTP 共享文件夹，家里有 NAS 的必备！没有的话，也可以用于在局域网下电脑和 Pad 互传文件。

ES 文件管理器直接访问家里 NAS

# 4. 速享

用于不同品牌的手机和平板间共享文件和文字。

速享 App，能发送文字，也能传文件

# 5. 一刻相册 / 时光相册

用于不同品牌的手机和平板间 “实时共享” 相册。挂载后台然后开启自动备份就可以了。记得换掉原系统相册的云备份功能，否则会出现冲突。

相册同步神器

# 6.Rotation

全局横屏软件，键鼠党必备。 只要你想横屏的 App，它都给你横屏。

全局横屏

以上就是我今天分享的一些安卓平板上使用的 App。它们大多把安卓的自由与开放性用到了极致。
