【软件推荐】一文详解UU安全以及其他安卓优化类软件

2020-06-29 原创教程 1.6k words
安卓

UU 安全服务器越来越不稳定了，想早一点选择一个替代产品的用户不妨现在动手开始实验其他软件。那么本帖将会给大家介绍大部分 UU 安全的替代产品。

如果大家有合适的软件也可以发布在本帖下面。

在介绍之前，我们先说明前提：

本帖介绍的所有产品必须工作在非 Root 模式下！

那些整天拿 Root 后的软件和非 Root 的软件比较的有什么意思？

那些整天吵着 Root 简单的有什么意思？你这么吊去把金立 GN3002 Root 了啊？

那些一言不合秀苹果的，来来来，少年你帮我破解个书旗小说，谢谢。

第一大类：原理和 UU 安全完全一致的软件：360 极客版，LBE 安全大师

很多人问 UU 安全什么原理，简单讲就是两个字：沙箱（SandBox），** 具体不解释，想知道的人可以百度一下。** 如果能找到相似的沙箱软件无疑是一个很不错的选择。

1.360 极客版

360 极客版沙箱模式可以创建一个程序与 UU 安全类似的绿色化版本，可以很好的管理应用权限。然而其操作手感很差，目前已知 BUG 是更换绿色版如果中途中断，则会包括原版软件在内一起消失。而且 360 已宣布放弃对 360 极客版的维护，所以并不是长久的选择。

2.LBE 安全大师

LBE 安全大师的沙箱模式很好用，拥有比 360 极客版更多的权限管理。然而！问题有两个：

首先支持软件较少，签名库没有 UU 安全和 360 极客版全。其次绿化后的软件没有桌面图标，每次都需要进入 LBE 安全大师打开，操作麻烦。但是如果信得过沙箱类别软件，LBE 安全大师可以考虑一下。

3.Island

来自绿色守护同作者的作品 lsland 是一款工作在手机的 “工作模式” 下的软件，其原理几乎等同于沙箱模式，工作模式下可以读取到一套完全不同的手机信息。软件小巧可靠，然而操作非常复杂，不适合初级玩家使用，而且国产各种阉割完的 Android 系统通常无法正常使用。但如果你是 Google 原版系统或者 CM 系列 ROM 不妨可以试一下。然而你都能刷 CM 了，Root 应该也不是什么难事了吧？

第二大类：另辟蹊径的 ADB 软件

众所周知，ADB 模式的权限奇高，仅次于 Root 模式，所以很多软件都想到用 ADB 模式管理权限。其中力推的就是两款：黑域，冰箱

\1. 黑域

黑域，ADB 模式下的最强后天管理利器，可以限制后台，可以禁止自启，没有什么能够阻挡，唯独缺点：每次重启后需要重新连接电脑进行设置，这个可能是一个巨大的问题。但这依旧不妨碍其成为最好用的免 Root 自启，后台以及切断唤醒软件。

\2. 冰箱

冻结应用神软，原来只能工作在 Root 模式下，现在也可以工作在 ADB 模式下了。冻结后的应用相当于系统自带的停用功能，简单说和卸载了差不多，而且不丢失数据。唯一缺点是授权麻烦，上手难度高。但是功能确实好用，强推。

第三大类：辅助功能停止软件

这个就很常见的一个类别了，大部分免 Root 的软件都工作在这个模式下，这里推荐的有两款。

\1. 净化大师

毫无疑问，净化大师是优化软件中做的相当出色的一款软件。其 Root 模式下可以很完美的完成自启管理，唤醒切断等功能，还能免疫部分软件广告。在非 Root 模式下，使用辅助功能停用软件，也是很方便的设计，使用 VPN 功能可以屏蔽广告。但其没有锁屏自动清理功能，实在是一大败笔，且后续版本体积很大，已经不再实用了。

\2. 绿色守护

绿色守护，老牌优化神软，我的所有能 Root 的设备上几乎都装的软件。其免 Root 模式下使用的是辅助功能加强行停止，很废。但是相比净化大师，其多了一个锁屏自动清理功能，少了一个 VPN 屏蔽广告功能。其还有一个劣势就是其很难保证自己在后台运行（这一点国产流氓做的更好一些）。尽管如此，干净小巧依旧是推荐这款软件的理由。

废了这么大劲写这么多到底是干嘛呢？

首要原因无外乎，UU 安全要停用了，说不准哪天，其突然就永久关闭服务器，留下我们一群被国产 BAT 全家桶欺负的手机用户，到时候再找替代软件岂不是会手忙脚乱。

安卓其实是一个非常好用的系统，只是被国产的无良商家搞臭了而已。前不久尝试把所有安卓软件替换成类似功能的国外软件，系统非常流畅，1G 内存完美运行，所以安卓在中国不好用绝对是中国的软件商老板逼中国程序员干的。

再次，现在很多人有一种浮躁之风，一言不合就扔软件扔观点，一点理由没有，“这个软件完爆了 UU 安全”，呸！我上面分析了多少款软件，哪个没有缺点？怎么就完爆了？

再者说，UU 难道就没缺点？部分软件商店无法直接唤醒 UU 不是缺点？UU 无法使用安卓提供的 “默认软件” 的方式替代系统自带安装器难道不是缺点？

希望大家养成踏实的 “玩机” 风气。尽可能设计和发现更多好用的软件给大家。
