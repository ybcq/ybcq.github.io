【独家汉化】Ardublock全功能汉化版

2020-06-29 独家修改 389 words
arduino

# 简介

ArduBlock 是一款开源的 Arduino 图形化编程软件，由上海新车间创客开发，是 Arduino 官方编程环境的第三方软件，必须依附于 Arduino ide 运行。

网上的 ArduBlock 中含有大量的英文内容，不适合 0 基础的网友编程，所以我制作了这个汉化版。

打开软件，工具，ArduBlock 进入！

# 软件截图

# 修改内容

1. 集成了 Arduino1.6.5，Ardublcok 版本更新为 20190210-Final
2. 基于 Ardublock 最完整功能版进行汉化，汉化覆盖率高达 99%(其中除 panda1~5 不能理解是什么意思，故未汉化)
3. 请确认您自己知道如何打开 Ardublock 软件
4. 如果打开不是中文，请在 Arduino 的设置里把语言设置成中文
5. 使用 7z 封包，右键解压即可当作绿色版使用，双击即可当作安装版使用

# 技术分享

jar 文件的汉化主要是对 xml 文件进行汉化。在这个软件的 xml 文件里，所有的汉字被写成了内码格式，所以对于此软件的汉化费时的点在于对内码进行转换。

# 注意事项

由于初琴使用的打包工具的 BUG，不能在卸载时自动清理注册表，所以推荐卸载后使用 Wise Care 365 清理注册表。
