2020-06-29 独家修改 350 words
浏览器

# 简介

不完全汉化，HTML 已汉化，但 JS 文件未汉化（汉化后功能失效），但已不影响平时使用。

安装后如不是中文，则在右上角→Configuration→最下面 Languages 选择 "汉化"，重启浏览器即可完成汉化。

右上角可以切换默认引擎为 Baidu，不过更推荐使用 Bing 搜索。

Aurora 浏览器绝对是我用过的最好的 IE 内核浏览器了，包含了手势操作，ADBlock，网页翻译等所有功能，界面简洁大方。

适合只能使用 IE 内核浏览器的小伙伴。

已知 BUG：安装在 C 盘时，如果不以管理员身份启动，则不能正确使用收藏夹和历史记录。
