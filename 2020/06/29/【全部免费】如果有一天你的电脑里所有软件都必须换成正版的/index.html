【全部免费】如果有一天你的电脑里所有软件都必须换成正版的

2020-06-29 免费软件 1.1k words
Windows

如果有一天，全面禁止了破解软件，要求你电脑里所有的软件都是正版的

那我也不会用思杰马克丁公司的软件！

不过现在倒是可以想想有哪些软件是依旧可以使用的吧？

首先，根据法律法规，以下两种情况不涉及侵权：

1. 个人使用且不传播自己已下载的被破解软件的副本。

2. 使用现在已收费但曾经免费的软件的旧版副本。

所以基于这两条规律，我总结了以下可以使用的内容：

1. 操作系统

Windows 作为一款闭源操作系统，一直是全球最大用户量的系统没有之一。中国的破解 Windows 副本使用量巨大。但以下情况不属于侵权：

购买电脑时该电脑已包含正版 Windows 标签；

以此为基础升级为 Windows10；

未使用非正常手段升级到其他版本（比如使用破解码从家庭版升级为旗舰版）；

以以上三条为基础由于系统损坏而重新刷同版本系统并破解系统达到正版。

基于以上条件，大家可以看一下自己的笔记本上有没有正版 Windows 标签，如果有，则恭喜你可以一直使用这个版本以及通过正常渠道升级的 Windows10 版本了。

不过通常情况下，组装台式机并没有预装正版 Windows，所以替换系统目前（家用或者办公用）推荐以下两个：

DEEPIN OS

集成了 Wine 商业版的 DEEPIN 可以兼容部分 Windows 软件，作为 Windows 系统的日常替代可以尝试，笔者正在使用，各方面兼容性尚可。

Gmac OS

如果你沉迷于 MAC 优秀的 UI 设计，Gmac 不妨是一个不错的尝试。内置的基本办公工具也可以满足日常的需求。

2.Office 篇

相比于其他软件，Office 的替代品是相当多的。

如果你是 Windows 7 用户，那你有福了，一个 MicroSoft 官方的软件 MicroSoft Office Starter 完全可以满足你的日常要求，不过这个软件不允许在 Windows 8 及以上的版本安装。

WPS Office，作为金山公司安装量最大的产品，WPS 可以说是国产最优秀的 Office 替代品没有之一。

OfficeSuite，免费开源的跨平台 Office 软件，安卓版和 IOS 版都非常好用，界面漂亮，功能尚可，但是 Windows 版的兼容性奇差。

MicroWord，功能简单，体积小的 Word 处理软件，功能类似于 Office2003。

LibreOffice，免费开源的跨平台 Office 软件，其 Linux 版本非常好用，功能强大。但是 Windows 版用户量小。

3.PS 篇

GIMP (Gnu Image Manipulation Program) 是适配于所有 OS（操作系统）的图像编辑工具。它不仅能免费下载，还具备了修饰图片所必需的功能、是以高性能为卖点的。但是，同 Photoshop 一样功能有些过于繁杂，界面交互设计方面也有待改善，要习惯使用它需要花一些时间。GIMP 有一个巨大的优势就是其兼容 PSD 格式的文件。

Pixir（Photoshop 网页版）

Pixlr 是一款可在线随时使用的云储存图像编辑工具。界面和 Photoshop 几乎一样，也有一些裁切工具、蒙板之类熟悉的功能。可以从本地上传图片进行编辑，适用于快捷处理图像。

NeoImaging（光影魔术手）

如果你只是想给照片调色加滤镜之类的简单操作，NeoImaging 光影魔术手将会是一个特别优秀的选择。

4.CAD 篇

CAD 替代软件比较少，推荐的是中望 CAD，现在关注微信号还可以领取正版激活码。

如果你只是平时画一画房间的平面图，CAD 迷你画图就能满足你的要求。

5、音频类

越是专业的软件免费替代软件就越少。

音频处理类软件 Adobe Audition 的替代软件比较少，如果不是做特别复杂的音频处理，可以推荐 Audacity：

Guitar Pro 和 Overture 一类的制谱软件最推荐的替代品还是 MuseScore，免费好用，功能不比专业软件差。
