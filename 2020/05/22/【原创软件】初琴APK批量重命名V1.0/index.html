【原创软件】初琴APK批量重命名V1.0

2020年05月22日

# 简介

有些人就喜欢闲来无事屯各种软件的安装包（比如站长）。站长有一个 Android 文件夹，内有很多 APK 文件。保存的时候没有命名，时间长了就找不到他们叫什么了。因此，站长完善了网上的一个软件，做成了一款 * 的 APK 批量重命名小软件。它可以把你的 APK 包统一命名为 “中文名 + 版本号” 的形式。

# 软件截图

# 源码分享

# 主要功能

1. 可以添加右键菜单

2. 可以关联 APK 文件

3. 使用拖拽的方式可以批量重命名 APK。不要用右键批量重命名！

# 注意事项

由于初琴使用的打包工具的 BUG，不在卸载时自动清理注册表，所以推荐卸载后使用 Wise Care 365 清理注册表。

由于使用了 BAT 部署，所以有可能杀软会报毒。不放心的可以自己解包看 BAT 部署代码。
